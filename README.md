🚀 Task Tracker

A modern and interactive Task Tracker (Todo List) application that helps you manage your daily tasks efficiently with a clean UI and smooth drag-and-drop experience.

✨ Features

📝 Create Tasks

Add new todo items 

Include detailed descriptions for each task

🔍 Task Details Modal

Open a modal to view and edit full task information

Add explanations or notes for better clarity

⚡ Priority Label

Mark tasks as urgent

Visually highlights high-priority items

📊 Kanban Board Layout

Organize tasks into 3 columns:

Todo

In Progress (Action)

Done

🔄 Drag & Drop

Smoothly move tasks:

Between columns

Reorder within the same column

Built using dnd-kit

💾 Local Storage Persistence

Tasks are saved in the browser

No backend required

🎨 Modern UI

Styled with Tailwind CSS

Clean, responsive, and user-friendly design

🛠️ Tech Stack

Frontend: React (or your framework)

Styling: Tailwind CSS

Drag & Drop: dnd-kit

State Persistence: Local Storage

Package Manager: Yarn

📦 Installation & Setup

Clone the repository:

git clone https://github.com/your-username/task-tracker.git

Navigate into the project:

cd task-tracker

Install dependencies:

yarn install

Start the development server:

yarn start

Add a new task using the input field

Click on a task to open the modal and add details

Use the priority button to mark urgent tasks

Drag and drop tasks between columns to update their status

Your tasks will automatically persist in local storage
